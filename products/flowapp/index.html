FAQ (Frequently Asked Questions)

Apakah saya memerlukan akun Google?

Ya. FlowApp memanfaatkan Google Sheets sebagai database Anda. Kode Google Apps Script yang di-generate harus di-deploy di dalam editor Google Apps Script Anda sendiri untuk akses penuh.

Apakah database aman?

Sangat aman. Database aplikasi Anda disimpan di Google Sheet milik akun Google Anda sendiri. FlowApp tidak memyimpan salinan record database Anda, melainkan hanya merender interface yang berinteraksi langsung via GAS Web App URL Anda.

Bagaimana cara memotong branding banner?

Paket Pro dan Cloud memberikan fitur khusus untuk menghapus branding banner "Built with FlowApp" di bagian bawah layar secara otomatis saat proses publikasi.
